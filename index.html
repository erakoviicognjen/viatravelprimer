How do I add or remove the preloader?

Right below the 

<div id="content">
tag comes the preloader code. The code within the preloader brackets needs to be added or removed to have the preloader active. The JavaScript (starter.js) is already included on all pages.

How to create another subpage?

Simply use one of the existing subpages (blog.html or post.html) and alter it to your specifications. If you need help, don't hesitate to leave a ticket on our support forum.

How to add columns?

Here you can utilize the grid system of bootstrap, which is fully integrated. For more information on how this system works, please take a look at http://getbootstrap.com/css/#grid

How to implement a slideshow?

Link the flexslider JS and CSS file from project.html and add in the designated content area:

<div class="flexslider">
<div class="images">
<ul class="slides">
<li><img src="images/image01.jpg" alt="" /></li> 
<li><img src="images/image02.jpg" alt="" /></li>
</ul> </div>
</div>

How to add fancybox lightbox and create groups of images/videos?

Wrap this around the image(s) or icon(s) where you want to add a larger image opening in lightbox upon mouseclick: For single Image: 
<a href="images/pic-large.jpg" data-title="" class="popup"></a>
For a group of images: 
<a href="images/pic-large.jpg" data-title="Image Title here"
data-fancybox-group="group1" class="popup"></a>
For a video popup: 
<a href="http://player.vimeo.com/video/48792900?title=0&byline=0&portrait=0&color=ffff ff" data-title="" class="video-popup"></a>

How to change the email address to which emails are sent from the booking/reservation form?

Find in reservation.php: $address = 'hello@email.com';
There you can enter your email address. A few lines further down, you can also change the text which is displayed in the email.

How to change the address of the integrated google maps?

Find in googlemaps.js line 47 and 64: You need to replace the coordinates (longitude, latitude) with your own location.
